# Notification Sounds

This directory contains notification sound files for the chat system.

## Sound Files (in priority order):

1. **tune.mp3** - Primary notification sound (user's custom sound)
2. **notification.mp3** - Default notification sound
3. **notification1.mp3** - Fallback notification sound 1
4. **notification2.mp3** - Fallback notification sound 2
5. **notification3.mp3** - Fallback notification sound 3
6. **alert.mp3** - Alternative alert sound
7. **chime.mp3** - Alternative chime sound
8. **ding.mp3** - Alternative ding sound

The system will automatically try these sounds in order until it finds one that works.

## Recommended Sources for Notification Sounds:

1. **Mixkit** - https://mixkit.co/free-sound-effects/notification/
   - Free notification sounds in MP3 format
   - No attribution required

2. **Freesound** - https://freesound.org/
   - Search for "notification" or "alert"
   - Check license before use

3. **Zapsplat** - https://www.zapsplat.com/
   - Free UI Kit sound pack (registration required)
   - 120+ interface sounds

## File Requirements:
- Format: MP3
- Duration: 0.5-2 seconds recommended (short, alert-like sounds)
- Volume: Normalized (not too loud or quiet)
- Size: Keep under 500KB for fast loading

## Installation:
1. Download notification sound files from the sources above
2. Rename them according to the priority list (tune.mp3, notification.mp3, etc.)
3. Place them in this directory (`client/public/audio/`)
